🎬 Seedance 2.0 视频生成

基于火山方舟 contents/generations/tasks API 的静态单页工具，支持 Seedance 2.0 / 2.0 Fast

📂 素材

  • 图片 / 音频：支持本地上传，自动转为 Base64。
  • 参考视频：仅支持方舟可访问的公网 https:// 或 asset:// 地址（接口在火山侧拉取，localhost / 局域网 IP 不可用）。
  • 音频不可单独使用，需至少搭配一张图片或一条参考视频 URL。
