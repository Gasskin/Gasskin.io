素材

图片与音频可本地上传（Base64）。参考视频须为方舟云端能访问的 URL（公网 https:// 或 asset://）：接口在火山侧拉取视频，不是你的浏览器。localhost、127.0.0.1、局域网 IP、file:// 对方都访问不到。若只有本机文件，可先上传到对象存储，或用 ngrok / Cloudflare Tunnel 等把本机 HTTP 临时暴露成公网 HTTPS 再填该地址。音频不可单独使用，需至少一张图或一条参考视频 URL。
